# Crop Type Classification 
## Using Multinomial Logistic Regression

The dataset was crated by augmenting existing rainfall, climate, and fertilizer datasets from India. As a result, the data is not pure raw observed information, rather it is a combination of source and constructed data. The data can be found in the following link: [Crop Recommendation Dataset](https://www.kaggle.com/datasets/atharvaingle/crop-recommendation-dataset "Crop Recommendation Dataset")
